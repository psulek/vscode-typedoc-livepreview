# log

```ts
(
  level, 
  message, 
  error?): void
```

Logs message and/or error at specified log level.

## Parameters


| Parameter | Type |
| :------ | :------ |
| `level` | `NonNullable`\< [`LogLevel`](type-alias.LogLevel.md) \> |
| `message` | `string` |
| `error`? | `Error` |


## Returns

`void`
