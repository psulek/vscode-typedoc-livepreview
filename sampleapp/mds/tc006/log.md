# Property: log

```ts
(
  level, 
  message, 
  error?): void
```

Logs message and/or error at specified log level.

## Parameters


| Parameter | Type | Description |
| :------ | :------ | :------ |
| `level` | `string` | log level |
| `message` | `string` | log message |
| `error`? | `Error` | log error (optional) |


## Returns

`void`
